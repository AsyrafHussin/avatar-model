# Avatar Model

Embed 3D Model using [model-viewer](https://modelviewer.dev/)

## Getting Started

```bash
# Clone the repo
git clone https://github.com/AsyrafHussin/avatar-model.git

# Navigate to project folder
cd avatar-model

# Initial Setup
npm install

# Serve the application on http://127.0.0.1:5173
npm run dev
```
